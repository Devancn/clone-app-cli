# clone-app

Clone a git repositort via shell command.

## Installation

```sh
$ yarn global add clone-app
```
> or

```sh
$ npm install -g clone-app
```

## Commands
```sh
#help
$ clone-app -h

# add a new template
$ clone-app add

# initial project from template
$ clone-app init

# view template name list
$ clone-app list

# remove template
$ clone-app remove
```
